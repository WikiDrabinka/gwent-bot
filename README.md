# 🎴Gwent Bot

This project's goal is to create a few different Reinforcement Learning Agents focused on playing Gwent, a game introduced in The Witcher 3, and eventually create a user interface where anyone could play against and test those agents.

## Road map

- [ ] Game functionality
    - [x] Basic cards and game
    - [ ] Leader cards
    - [ ] Expansion pack abilities and cards
- [ ] Agents
    - [ ] Baseline naive algorithm
    - [ ] Q-Learning Agent
    - [ ] PPO Agent
- [ ] User interface
    - [ ] Fully operable console game
